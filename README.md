# backend-learning-archive

This repository contains my backend learning projects. I use it to practice Node.js API development and improve code structure with each project.

## Projects

### 1. contact-manager-nodejs-application
Learning focus:
- REST APIs with Express
- MongoDB + Mongoose integration
- Authentication with JWT
- Input validation and middleware-based error handling

Tech:
- Node.js
- Express 5
- MongoDB
- JWT
- Zod

Run:
```bash
cd contact-manager-nodejs-application
npm install
npm run dev
```

### 2. short-url-nodejs-main
Learning focus:
- URL shortening workflow
- Server-side rendering with EJS
- Route and service separation
- Basic cookie handling and redirects

Tech:
- Node.js
- Express
- MongoDB
- EJS

Run:
```bash
cd short-url-nodejs-main
npm install
npm start
```
